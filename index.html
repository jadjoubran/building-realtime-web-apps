Step 3

Backend setup

  • Open the Pusher Documentation

    Open pusher.com/docs and locate the API libraries and then scroll through to find the PHP library.

  • Composer install it

    From the CLI:

    composer require pusher/pusher-php-server
  • Require autoloader

    Open server.php and require the composer autoloader:

    require("vendor/autoload.php");
  • Pusher Constructor

    Open pusher/pusher-http-php on github and use the code to create a new Pusher instance in your server.php.
    Don't forget to use your credentials which are provided in pusher's App Keys page.

  • Testing Client Side Integration

    Open your Pusher App Keys page and then go to server.php and trigger an the following using the Pusher constructor you just created:

    Channel test_channel
    Event my_event
    Data ["message": "Hello World"]

    Navigate to server.php page in your brwoser and expect an Alert in the App Keys page!
